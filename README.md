# Registro de Chamados – Ambulância Guaiçara

- HTML/CSS/JS puro com Firebase Auth + Realtime Database
- Tabela em tempo real, chat interno, replicar/excluir chamados

## Configuração
1. Crie um projeto Firebase.
2. Ative Authentication (Email/Password).
3. Ative Realtime Database (modo seguro).
4. Copie o config para `public/login.html` e `src/main.js`.

## Deploy
- Suba ao GitHub.
- Conecte no Vercel.
- Use o domínio de produção `.vercel.app`.
